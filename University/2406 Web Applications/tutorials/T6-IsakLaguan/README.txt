TO RUN APP
    - after extraction
    - change directories till youre within the tutorial folder in terminal
    - make sure to have node.js already installed
    - install all dependencies with
        - npm install
    - to run the server
        - node recipeserver.js
    - play around with app functionality on chrome at http://localhost:3000/ or http://localhost:3000/index.html to start
