TO RUN APP
    - after extraction
    - change directories till youre within the tutorial folder in terminal
    - make sure to have node.js already installed
    - install pug and dependencies with 
        - npm install pug
        - npm init --yes
    - to run the server
        - node server.js
    - play around with app functionality on chrome at http://localhost:3000/cards to start
        - happy marking :)
